## Color schema

The White-Gold Gradient Color Scheme palette has 5 colors which are **Floral White** (#FFFAF1), **Dutch White** (#EDDDB8), **Burlywood** (#DCC080), **Aztec Gold** (#CAA347) and **Dark Goldenrod** (#B8860E).
